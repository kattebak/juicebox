🧃 as-me — create your scoped GitHub App

This page generates a manifest with the locked permissions below and submits it to GitHub. You'll review the App on github.com and click Create GitHub App.

Must be globally unique on GitHub. ${USER}-only is the suggested default.

Shown on the App's public GitHub page. Not security-critical.

Personal = the App lives under your account, only you install/manage it. Org = lives under the org; teammates with admin can install/manage too.

Permissions granted

  • Contents — read & write
  • Pull requests — read & write
  • Issues — read & write
  • Statuses — read & write
  • Metadata — read
  • Administration, Members, Actions, Packages, Secrets, Pages, Workflows — no access

This is the whole juicebox. Locked by the manifest — you can't expand the permissions on this page, and a compromised agent can never do anything outside this list.

After clicking, you'll land on github.com to confirm. Then you'll be redirected to a small "paste this back" page with the code you need for your terminal.
